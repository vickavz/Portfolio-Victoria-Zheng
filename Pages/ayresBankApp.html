Ayres Bank Website Design

As part of my Google UX Design Professional course, my second project started out by choosing a topic using the tool Sharpen to generate a random design challenge. I was given the following task: "Create a balance transfer flow for a bank". This is a solo case study for a hypothetical business, so I came up with the idea to create a age and user-friendly online banking website to help mainly senior citizens to manage their finance.
