G-Full Web Redesign G-Full Website Redesign

This is a UX/UI redesign project of an e-commerce website (mobile and desktop version) for a Chinese company G-Full S.A., which I worked with a web developer to carry it out. I realized that the UI of their web was not very user-friendly and it was quite difficult to understand the whole commodity order process. That's why I came up with the idea to redesign it, to make it simpler and easier to use, in order to improve the user experience. I am sure that a redesign would help the web grow much better than before. The website is currently in the development process.
