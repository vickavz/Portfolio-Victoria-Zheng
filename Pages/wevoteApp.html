WeVote App

As part of my Google UX Design Professional course, my last project started out by choosing a topic using the tool Sharpen to generate a random design challenge focused on creating social good. I was given the task of "Design a scalable tool to help people to register to vote". In this project I designed a dedicated mobile app and a complementary responsive website to help students participate remotely in university elections.
